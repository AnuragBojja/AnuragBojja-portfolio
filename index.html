🏗️ 10-Layer IaC Architecture
Infrastructure as Code

Production Microservices Infrastructure (10-Layer IaC Architecture)

Architected a fully automated, zero-touch production environment across 10 independently managed infrastructure layers, where each layer is isolated with its own remote state — eliminating cross-layer dependencies and enabling teams to modify any layer without risk to the rest of the system.

  • Enforced security at the infrastructure level — least-privilege network policies, centralized secrets management with no credentials in code, and TLS encryption at the edge.
  • Engineered a self-bootstrapping entry point that automatically provisions databases, bakes service images, configures all services, and deploys behind an auto-scaling load-balanced architecture — fully hands-free from first boot to production.
  • Layered Ansible configuration management on top, where any service deploys through a single parameterized command using Jinja2 templates and runtime secret resolution.

☸️ Kubernetes Orchestration Platform
Containers & Orchestration

Containerized Microservices Platform (Kubernetes Orchestration & Helm Packaging)

Containerized and orchestrated a 10-service polyglot microservices platform end-to-end — from optimized Docker images through production-grade Kubernetes deployments to reusable Helm charts.

  • Built optimized Docker images for all 10 services using multi-stage builds, Alpine base images, and non-root user enforcement — eliminating build-time tooling from runtime containers.
  • Deployed services to a managed Kubernetes cluster using Deployments for stateless workloads, StatefulSets with persistent storage for databases, ConfigMaps for environment-driven configuration, and HPA for traffic-driven scaling.
  • Implemented security and reliability controls — RBAC with role and role-binding configurations, IRSA-based pod-to-cloud authentication via OIDC, and liveness/readiness probes with appropriate thresholds per service type.
  • Packaged the entire deployment as reusable Helm charts with values-driven templating, enabling environment-specific deployments without code duplication.
